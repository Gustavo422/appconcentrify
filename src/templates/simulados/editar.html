Formato do arquivo JSON

  • O arquivo deve conter um objeto JSON válido
  • Estrutura básica: 
    {
    "titulo": "Título do Simulado",
    "descricao": "Descrição do simulado",
    "areas_conhecimento": [
        {
            "materia": "Nome da Matéria",
            "questoes": [
                {
                    "enunciado": "Texto da questão",
                    "alternativas": {
                        "A": "Alternativa A",
                        "B": "Alternativa B",
                        "C": "Alternativa C",
                        "D": "Alternativa D",
                        "E": "Alternativa E"
                    },
                    "correta": "A"
                }
            ]
        }
    ]
}
  • Todas as questões devem ter 5 alternativas (A a E)
  • A alternativa correta deve ser uma das letras (A, B, C, D ou E)
